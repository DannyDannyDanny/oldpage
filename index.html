Hosting python server for local dev:

  • cd to the local dannydannydanny.github.io directory

    cd "$(find . -name dannydannydanny.github.io | head -1)"

  • Python 2.x: $ run python -m SimpleHTTPServer 8890

  • Python 3.x: $ python3 -m http.server 8890
